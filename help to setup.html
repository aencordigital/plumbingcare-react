EmailJS Setup Guide

Follow these steps to configure EmailJS for your PlumbingCare app

Create Your EmailJS Account

1

Sign Up for EmailJS

  1. Go to the EmailJS website
  2. Click on "Sign Up" in the top right corner
  3. Choose to sign up with Google, GitHub, or your email address
  4. Complete the registration process
  5. Verify your email address if required

EmailJS offers a free plan that includes 200 monthly emails, which is perfect for testing and small applications.

2

Access Your Dashboard

After signing up, you'll be taken to your EmailJS dashboard. This is where you'll manage your email services and templates.

Dashboard → Email Services → Email Templates → Account

Familiarize yourself with the dashboard navigation, as you'll need to access these sections in the following steps.

Set Up Your Email Service

1

Add an Email Service

  1. In your EmailJS dashboard, click on "Email Services"
  2. Click the "Add New Service" button
  3. Select your email service provider

For testing, Gmail is recommended. For production, consider using a transactional email service like SendGrid or Mailjet.

2

Connect Gmail (Recommended for Testing)

  1. Select "Gmail" as your service provider
  2. Click "Connect Account"
  3. Sign in to your Gmail account when prompted
  4. Grant the necessary permissions to EmailJS

If you have 2-factor authentication enabled on Gmail, you may need to create an App Password for EmailJS.

3

Note Your Service ID

After connecting your email service, you'll see a Service ID. This is a unique identifier for your email service.

Service ID: service_abc123xyz

You'll need this Service ID for your environment variables later.

Create Email Templates

1

Create a New Template

  1. In your EmailJS dashboard, click on "Email Templates"
  2. Click the "Create New Template" button
  3. Give your template a name (e.g., "Service Confirmation")
2

Design Your Template

Use the template editor to design your email. For the PlumbingCare app, you'll want to include these dynamic variables:

{{to_name}}          // Customer's name
{{service}}         // Requested service
{{notes}}           // Additional notes
{{emergency}}       // Emergency flag
{{preferred_date}}  // Preferred service date

These variables will be automatically replaced with actual values when the email is sent.

3

Note Your Template ID

After saving your template, you'll see a Template ID. This is a unique identifier for your email template.

Template ID: template_abc123xyz

You'll need this Template ID for your environment variables.

Integrate EmailJS with Your App

1

Get Your User ID

  1. In your EmailJS dashboard, click on "Account"
  2. Find your User ID in the account info section
  3. Copy your User ID
User ID: user_abc123xyz
2

Set Up Environment Variables

Create a .env file in your React project root and add these variables:

REACT_APP_EMAILJS_SERVICE_ID=your_service_id_here
REACT_APP_EMAILJS_TEMPLATE_ID=your_template_id_here
REACT_APP_EMAILJS_USER_ID=your_user_id_here

Replace the values with your actual Service ID, Template ID, and User ID from EmailJS.

3

Test Your Configuration

Send a test email to verify your setup is working correctly:

// In your service request function
emailjs.send(
  process.env.REACT_APP_EMAILJS_SERVICE_ID,
  process.env.REACT_APP_EMAILJS_TEMPLATE_ID,
  {
    to_name: customerName,
    service: serviceType,
    notes: serviceNotes,
    emergency: isEmergency ? 'Yes' : 'No',
    preferred_date: preferredDate
  },
  process.env.REACT_APP_EMAILJS_USER_ID
)
.then(() => {
  console.log('Email sent successfully!');
})
.catch(error => {
  console.error('Email sending failed:', error);
});

Setup Complete!

You've successfully set up EmailJS for your PlumbingCare application. Your app can now send automated email notifications to customers when they request services.
